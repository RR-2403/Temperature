# Temp-Conv

A simple temperature converter web app that converts between Celsius and Fahrenheit.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Clean, minimal UI with Bootstrap styling
- Instant result displayed on the page

## How to Use

1. Open `index.html` in your browser.
2. Enter a numeric temperature in the input field.
3. Select one of the conversion options:
   - `Celsius ➡️ Fahrenheit`
   - `Fahrenheit ➡️ Celsius`
4. Click the `Convert` button.
5. The converted temperature appears below the button.

## Files

- `index.html` - app structure and form UI
- `style.css` - custom styles for the page
- `app.js` - conversion logic and result display

## Notes

- The converter rounds results to two decimal places.
- Make sure JavaScript is enabled in your browser.
